# Geometry datasets to shapefile format

This solution allows to convert a shapefile coming from a dataframe or a geodataframe

# Shp class

## Methods

|method|descrption|
|-|-|
|**is_geodataframe**|Evaluates if the object is a `GeoDataFrame`|
|**build_geodataframe**|Converts a `DataFrame` to a `GeoDataFrame`|
|**export_shp**|Exports `GeoDataFrame` to a `Shapefile` format|
